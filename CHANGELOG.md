# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-01-09

### Added
- **Checksum Calculator**: New tool with 11 algorithms (CRC-8, CRC-16, CRC-32, Adler-32, MD5, SHA-1, SHA-256, SHA-384, SHA-512, SHA3-256, SHA3-512)
- **Double-click Popup**: View full checksum in popup dialog for long hashes
- **Drag & Drop**: Drop files directly onto the checksum calculator
- **Enhanced Scrollbars**: Improved scrollbar styling with gradient handles and hover effects

## [1.0.0] - 2026-01-08

### Added
- **Centralized Versioning**: Improved internal version handling.
- **Auto-Updater**: Integrated auto-update capability to fetch the latest release from GitHub.
- **UI Versioning**: Displayed application version in the main window title.
- **Icon Update**: Updated UDP protocol icon in documentation.
